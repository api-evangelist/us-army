# US Army

The US Army is the largest branch of the United States military, responsible for conducting ground combat operations and defending the nation. The Army maintains a public REST API at api.army.mil using the OpenAPI 3 (OAS3) specification, providing access to Army news articles, press releases, and public events. The Army also maintains open source projects through the US Army Research Laboratory GitHub organization and contributes to the DoD open source ecosystem at Code.mil.
